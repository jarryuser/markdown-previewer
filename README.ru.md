## Обзор

Markdown Previewer - это редактор на основе браузера с расположением панелей side-by-side. Пишите Markdown слева, а результат отображается справа в реальном времени. Выходной рендеринг соответствует GitHub: GFM-таблицы, списки задач с собственными чекбоксами и блоки кода с подсветкой синтаксиса.

Также в проект входит небольшой CLI (`mdp`), который отображает любой локальный `.md` файл в браузере с теми же стилями и автоматически перезагружает страницу при сохранении файла.

## CLI - `mdp`

`mdp` работает в двух режимах:

- **Режим файла** - открывает один `.md` файл; страница автоматически перезагружается при каждом сохранении
- **Режим папки** - открывает папку с боковой панелью дерева файлов; нажмите на любой файл для предпросмотра; перезагружается при изменении текущего файла

Изображения, указанные по относительным путям, обслуживаются автоматически.

### Установка

```bash
# 1. clone the repo
git clone https://github.com/jarryuser/markdown-previewer.git
cd markdown-previewer

# 2. install dependencies
npm install

# 3. build the CLI bundle
npm run build:cli

# 4. register the command globally
npm link
```

### Использование

```bash
# single file
mdp README.md
mdp ~/notes/ideas.md

# entire directory (Obsidian vault, notes folder, etc.)
mdp ~/notes/
mdp ./docs/

# specify a fixed port
mdp README.md --port 3000

# show recent files (numbered list)
mdp

# reopen entry #2 from the recent list
mdp 2

# render to terminal with ANSI colors (stdout, no browser)
mdp README.md -t
mdp README.md --terminal

# interactive TUI: split-pane in the terminal
mdp README.md --tui
mdp ~/notes/ --tui

# pick a color scheme for terminal/TUI output (dark, light, sepia, nord)
mdp README.md -t --theme nord
mdp ~/notes/ --tui --theme sepia
```

Браузер открывается автоматически. Редактируйте файлы в любом редакторе - страница перезагружается при каждом сохранении.

В режиме папки нажмите `/`, чтобы отфильтровать дерево файлов по имени, `j`/`k` для навигации и `c`, чтобы свернуть или развернуть все папки.

### Режим TUI

`--tui` открывает полноэкранный терминальный интерфейс вместо браузера. HTTP-сервер всё ещё запускается в фоне, поэтому нажатие `b` в любой момент открывает текущий файл в браузере.

| Клавиша | Действие |
|---|---|
| `j` / `k` | прокрутка предпросмотра или навигация по дереву |
| `Tab` | переключение фокуса между деревом и предпросмотром |
| `l` / `Enter` | открыть файл или развернуть папку |
| `h` | свернуть папку или перейти к родительской |
| `/` | поиск файлов по имени |
| `b` | открыть текущий файл в браузере |
| `q` / `Ctrl+C` | выход |

### Удаление

```bash
npm unlink -g markdown-previewer
```

## Технический стек

| Layer | Tool | Почему |
|---|---|---|
| Язык | TypeScript 5.3 | Безопасность типов, строгий режим включён |

## План развития

### Сделано

- [x] **CLI-просмотрщик** (`mdp`) - открывает любой `.md` файл в браузере с автоперезагрузкой; локальные изображения обслуживаются из папки файла
- [x] **Перетаскивание изображений** - перетащите изображения в редактор; вставляются как короткие плейсхолдеры `local://N` вместо base64
- [x] **Списки задач в стиле GitHub** - собственные чекбоксы без маркеров, как в GitHub
- [x] **Подсветка синтаксиса в редакторе** - CodeMirror 6 с поддержкой языка Markdown и тёмной/светлой темой
- [x] **Синхронизированная прокрутка** - кнопка Sync на панели инструментов синхронизирует позиции прокрутки редактора и предпросмотра
- [x] **Сохранение содержимого в localStorage** - содержимое автоматически сохраняется при обновлении страницы
- [x] **Переключение переноса строк** - кнопка Wrap включает/выключает перенос строк; позиция прокрутки восстанавливается после изменения макета
- [x] **Экспорт в HTML** - скачивает готовый предпросмотр как отдельный файл со всеми встроенными стилями
- [x] **Горячие клавиши форматирования** - `Mod+B` жирный, `Mod+I` курсив, `Mod+K` ссылка, `Mod+Shift+C` строчный код; повторное нажатие отменяет
- [x] **Имя файла при экспорте** - запрашивает имя файла перед скачиванием
- [x] **Регулировка размера шрифта** - кнопки A− / A+; сохраняется в localStorage
- [x] **Полноэкранный режим** - кнопка Full расширяет приложение на весь экран
- [x] **Режим Vim** - кнопка Vim включает модальное редактирование с режимами Normal / Insert / Visual и строкой состояния
- [x] **Математические формулы** - KaTeX рендерит формулы `$строчные$` и `$$блочные$$` в предпросмотре
- [x] **Режим папки** - `mdp ./notes/` открывает папку с боковой панелью дерева файлов; нажмите на любой `.md` файл для предпросмотра; автоперезагрузка при сохранении
- [x] **Поиск и замена** - кнопка Find и `Mod+F` открывают встроенную панель поиска и замены CodeMirror
- [x] **Содержание** - кнопка TOC показывает боковую панель с кликабельными заголовками, которые прокручивают предпросмотр
- [x] **Диаграммы Mermaid** - блоки кода ` ```mermaid ` рендерятся как диаграммы через mermaid.js; тема соответствует светлому/тёмному режиму
- [x] **Флаг `--port`** - указать фиксированный порт: `mdp README.md --port 3000`
- [x] **Недавние файлы** - `mdp` без аргументов показывает нумерованный список недавно открытых файлов; `mdp 2` открывает запись №2
- [x] **Поиск в дереве файлов** - нажмите `/` для фильтрации файлов по имени; `↑`/`↓` для навигации по результатам; `Enter` для открытия; `Escape` для закрытия
- [x] **Вики-ссылки** - `[[filename]]` и `[[filename|alias]]` в режиме папки навигируют между файлами в хранилище
- [x] **Открыть локальный файл (FSA)** - кнопка Open File загружает любой `.md` файл в редактор; страница автоматически перезагружается при внешнем сохранении; `Mod+S` сохраняет изменения на диск
- [x] **Открыть локальную папку (FSA)** - кнопка Open Dir открывает папку через File System Access API; полное дерево файлов на боковой панели, нажмите на любой файл для предпросмотра
- [x] **Навигация назад / вперёд** - история как в браузере для FSA-режима; `Alt+←` / `Alt+→` или кнопки со стрелками навигируют между просмотренными файлами
- [x] **Изменяемая боковая панель** - перетаскивайте ручку между панелью и содержимым для изменения размера; работает в FSA-режиме и в CLI-режиме папки
- [x] **Хлебные крошки** - путь к текущему файлу отображается над предпросмотром в режиме папки; каждый сегмент - кликабельная ссылка
- [x] **Вывод в терминал** (`-t` / `--terminal`) - рендерит любой `.md` файл в stdout с ANSI-цветами; заголовки, жирный, курсив, блоки кода, таблицы; можно передавать в `less`
- [x] **Режим TUI** (`--tui`) - интерактивный разделённый экран в терминале; дерево файлов слева, предпросмотр справа; навигация j/k, поиск `/`, `b` для открытия в браузере; автообновление при сохранении файла
- [x] **Вставка изображений из буфера обмена** - вставляйте изображения через `Ctrl+V`, сохраняются так же, как при перетаскивании
- [x] **Печать / экспорт в PDF** - кнопка Print открывает диалог печати браузера с CSS для печати, который скрывает редактор и оставляет только текст
- [x] **Пользовательские темы** - Светлая, Тёмная, Сепия и Nord; кнопка Theme переключает между ними, выбор сохраняется между перезагрузками
- [x] **Флаг `--theme`** - выбрать цветовую схему для вывода в терминал и TUI: `mdp README.md -t --theme nord`
- [x] **Эмодзи-шорткоды** - превращает `:smile:` в 😄 в предпросмотре, как в GitHub
- [x] **Сноски** - рендеринг синтаксиса сносок (`[^1]`) в предпросмотре
