## Огляд

Markdown Previewer - це браузерний редактор із розділеним макетом. Пишіть Markdown ліворуч, бачте згенерований результат праворуч у реальному часі. Результат відповідає рендерингу GitHub: таблиці GFM, списки завдань із власними прапорцями та блоки коду з підсвічуванням синтаксису.

Проєкт також включає невеликий CLI (`mdp`), який відображає будь-який локальний `.md` файл у браузері з тими самими стилями та перезавантажує сторінку щоразу, коли файл зберігається.

## CLI - `mdp`

`mdp` працює у двох режимах:

- **File mode** - відкриває один `.md` файл; сторінка автоматично оновлюється при кожному збереженні
- **Directory mode** - відкриває теку з бічною панеллю файлового дерева; клацніть будь-який файл для перегляду; оновлюється при зміні поточного файлу

Зображення, указані з відносними шляхами, обслуговуються автоматично

### Встановлення

```bash
# 1. clone the repo
git clone https://github.com/jarryuser/markdown-previewer.git
cd markdown-previewer

# 2. install dependencies
npm install

# 3. build the CLI bundle
npm run build:cli

# 4. register the command globally
npm link
```

### Використання

```bash
# single file
mdp README.md
mdp ~/notes/ideas.md

# entire directory (Obsidian vault, notes folder, etc.)
mdp ~/notes/
mdp ./docs/

# specify a fixed port
mdp README.md --port 3000

# show recent files (numbered list)
mdp

# reopen entry #2 from the recent list
mdp 2

# render to terminal with ANSI colors (stdout, no browser)
mdp README.md -t
mdp README.md --terminal

# interactive TUI: split-pane in the terminal
mdp README.md --tui
mdp ~/notes/ --tui

# pick a color scheme for terminal/TUI output (dark, light, sepia, nord)
mdp README.md -t --theme nord
mdp ~/notes/ --tui --theme sepia
```

Браузер відкривається автоматично. Редагуйте файли в будь-якому редакторі - сторінка оновлюється при кожному збереженні

У режимі теки натисніть `/` для фільтрації файлового дерева за назвою, `j`/`k` для навігації та `c` для згортання або розгортання всіх тек

### Режим TUI

`--tui` відкриває повноекранний термінальний інтерфейс замість браузера. HTTP-сервер все одно запускається у фоновому режимі, тому натискання `b` у будь-який момент відкриває поточний файл у браузері

| Key | Action |
|---|---|
| `j` / `k` | scroll preview or navigate tree |
| `Tab` | switch focus between tree and preview |
| `l` / `Enter` | open file or expand folder |
| `h` | collapse folder or jump to parent |
| `/` | search files by name |
| `b` | open current file in browser |
| `q` / `Ctrl+C` | quit |

### Видалення

```bash
npm unlink -g markdown-previewer
```

## Технологічний стек

| Layer | Tool | Why |
|---|---|---|

| Deploy | GitHub Pages via Actions | Автоматичне розгортання при push у `main` |

- [x] **Перетягування зображень** - перетягніть зображення в редактор; вставляються як короткі замінники `local://N` замість base64
- [x] **Списки завдань у стилі GitHub** - власні прапорці без маркерів списку, що відповідають рендерингу GitHub
- [x] **Підсвічування синтаксису в редакторі** - CodeMirror 6 із підтримкою мови Markdown та темною/світлою темою
- [x] **Синхронізоване прокручування** - кнопка Sync на панелі інструментів синхронізує позиції прокрутки редактора та попереднього перегляду
- [x] **Збереження вмісту в localStorage** - вміст автоматично переживає оновлення сторінки

- [x] **Експорт у HTML** - завантажує згенерований попередній перегляд як окремий файл з усіма вбудованими стилями
- [x] **Гарячі клавіші форматування** - `Mod+B` жирний, `Mod+I` курсив, `Mod+K` посилання, `Mod+Shift+C` вбудований код; повторне натискання скасовує
- [x] **Власне ім'я файлу для експорту** - пропонує змінити ім'я файлу перед завантаженням
- [x] **Регульований розмір шрифту** - кнопки A− / A+; зберігається в localStorage
- [x] **Повноекранний режим** - кнопка Full розгортає додаток на весь екран
- [x] **Клавіші Vim** - кнопка Vim перемикає модальне редагування з режимами Normal / Insert / Visual та рядком стану
- [x] **Підтримка математики** - KaTeX рендерить формули `$inline$` та `$$block$$` у попередньому перегляді
- [x] **Режим теки** - `mdp ./notes/` відкриває теку з бічною панеллю файлового дерева; клацніть будь-який `.md` файл для перегляду; автооновлення при збереженні
- [x] **Пошук та заміна** - кнопка Find та `Mod+F` відкривають вбудовану панель пошуку та заміни CodeMirror
- [x] **Зміст** - кнопка TOC показує бічну панель із клікабельними заголовками, які прокручують попередній перегляд
- [x] **Діаграми Mermaid** - блоки коду ` ```mermaid ` рендеряться як діаграми через mermaid.js; тема слідує світлому/темному режиму
- [x] **Прапорець `--port`** - вкажіть фіксований порт: `mdp README.md --port 3000`
- [x] **Нещодавні файли** - `mdp` без аргументів показує нумерований список нещодавно відкритих файлів; `mdp 2` знову відкриває запис №2
- [x] **Пошук у файловому дереві** - натисніть `/` для фільтрації файлів за назвою; `↑`/`↓` для навігації; `Enter` для відкриття; `Escape` для закриття
- [x] **Вікіпосилання** - посилання `[[filename]]` та `[[filename|alias]]` у режимі теки навігують між файлами в сховищі
- [x] **Відкриття локального файлу (FSA)** - кнопка Open File завантажує будь-який `.md` файл безпосередньо в редактор; сторінка автоматично оновлюється при зовнішньому збереженні файлу; `Mod+S` зберігає зміни на диск
- [x] **Відкриття локальної теки (FSA)** - кнопка Open Dir відкриває теку за допомогою File System Access API; повна бічна панель файлового дерева, клацніть будь-який файл для перегляду
- [x] **Навігація назад / вперед** - історія в стилі браузера в накладанні теки FSA; `Alt+←` / `Alt+→` або кнопки зі стрілками навігують між переглянутими файлами
- [x] **Змінюваний розмір бічної панелі** - перетягніть ручку між бічною панеллю та вмістом для зміни розміру; працює як у накладанні FSA, так і в режимі теки CLI
- [x] **Хлібні крихти** - шлях до поточного файлу показується над попереднім переглядом у режимі теки; кожен сегмент є клікабельним посиланням
- [x] **Термінальний вивід** (`-t` / `--terminal`) - виводить будь-який `.md` файл у stdout із кольорами ANSI; заголовки, жирний, курсив, блоки коду, таблиці; можна передавати в `less`
- [x] **Режим TUI** (`--tui`) - інтерактивний розділений перегляд у терміналі; файлове дерево ліворуч, попередній перегляд праворуч; навігація j/k, пошук `/`, `b` для відкриття в браузері; автооновлення при збереженні файлу
- [x] **Вставка зображень із буфера обміну** - вставляйте зображення через `Ctrl+V`, зберігаються так само, як і при перетягуванні
- [x] **Друк / експорт у PDF** - кнопка Print відкриває діалог друку браузера з CSS для друку, який ховає редактор і залишає лише текст
- [x] **Власні теми** - світла, темна, сепія та Nord; кнопка Theme циклічно перемикає їх, а вибір зберігається після перезавантажень
